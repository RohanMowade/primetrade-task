# PrimeTrade Task Manager API

## Overview
RESTful API for task management with JWT authentication and role-based access control.

## Tech Stack
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Bcrypt** - Password hashing
